# Tipos de Dados no TypeScript

## Estrutura de Estudos

1. **Básicos**:

   - String
   - Number
   - Boolean
   - Array
   - Tuple
   - Enum

2. **Complexos**:

   - Object
   - Union e Literal
   - Alias
   - Interfaces

3. **Avançados**:

   - Void
   - Never
   - Nullable
   - Intersection
   - Generics

4. **Utilitários**:
   - Readonly
   - Keyof e Record

Cada arquivo possui exemplos e explicações. Use `tsc <arquivo>.ts` para compilar e testar!
